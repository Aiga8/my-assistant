# Модуль 14: Регрессионный запуск автотестов

## Назначение

Модуль закрепляет место для **репортов регрессионного прогона** автотестов. В рамках модуля никаких действий не выполняется — он задаёт только маршрутизацию и **куда сохранять отчёты**.

## Где хранить репорты

Все репорты по регрессионному запуску автотестов хранятся в:

**`results/14-autotests-regression/`**

Сюда переносятся отчёты из папки `regress_reports` (HTML и прочие артефакты прогонов). Здесь же сохраняются новые репорты после каждого регрессионного прогона.

## Маршрутизация

- если команда про **регрессионный запуск автотестов** / **репорты регресса** / **хранение отчётов регрессионного прогона** → модуль `14-autotests-regression`

## Быстрый старт (копипаст)

```
Открой: modules/14-autotests-regression/README.md

Регрессионный прогон выполнен. Сохрани репорты в results/14-autotests-regression/.
```
