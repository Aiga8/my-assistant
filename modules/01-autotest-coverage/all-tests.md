### 3. Импорт BHA-компонентов: доступность, валидация файла, замена данных и обработка ошибок
Проверяется поведение импорта компонентов: доступность действия, валидация формата/размера файла, замена существующих данных и отображение сообщений об успехе/ошибках.

**TestRail:**
- ID: _(заполнить позже)_
- Теги: _(заполнить позже)_

**Предусловия:**
- Приложение DrillSpot доступно по базовому URL
- Доступен экран/раздел с таблицей компонентов BHA для выбранного Run

**Шаги:**
1. Открыть базовый URL приложения DrillSpot
2. Перейти к экрану с таблицей компонентов BHA выбранного Run и дождаться загрузки
3. Запустить импорт компонентов, выбрать файл и дождаться результата
4. Повторить импорт для поддерживаемых и неподдерживаемых форматов, а также для разных размеров файлов
5. Выполнить импорт при пустой таблице компонентов и убедиться, что данные появились
6. Выполнить импорт при наличии данных и подтвердить замену
7. Выполнить импорт, приводящий к ошибке, и проверить отображение сообщения об ошибке
8. Выполнить импорт перетаскиванием файла в область таблицы компонентов
9. Открыть экран пользователем с ограниченными правами и проверить доступность импорта

**Ожидаемый результат:**
- Страница приложения загружена (заголовок вкладки браузера содержит «DrillSpot»)
- Действие импорта доступно на экране компонентов
- Для поддерживаемых форматов и размеров до 30MB не отображается критическое уведомление об ошибке
- Для неподдерживаемого формата отображается сообщение «File format is not supported» и список поддерживаемых форматов
- Для размера файла больше 30MB отображается сообщение «File size is too large, please upload a file less than 30MB»
- После импорта в пустой список в таблице появляются строки компонентов
- При наличии компонентов перед заменой отображается предупреждение о замене с действиями подтверждения/отмены
- После подтверждения замены отображаются данные компонентов, а ранее существовавшие данные не сохраняются
- При успешном импорте отображается сообщение «Components successfully imported»
- При ошибке отображается критическое уведомление с текстом об ошибке
- Для пользователя с ограниченными правами действие импорта недоступно и отображается подсказка «Insufficient permission»

**Автотест:**
- Название теста: Complete BHA import functionality test
- Путь: playwright/tests/bha-import/bha-import.spec.ts
- Ссылка: ссылка не предоставлена

**Примечание (опционально):**
В коде набор тестов помечен как пропущенный (временно не исполняется при запуске).
