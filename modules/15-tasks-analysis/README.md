# Модуль 15: Результаты анализа и работы по таскам

## Назначение

Модуль закрепляет место для **результатов анализа и работы по отдельным таскам** (Jira, QAA и др.). В рамках модуля никаких действий не выполняется — он задаёт только маршрутизацию и **куда сохранять артефакты**.

## Где хранить результаты

Все результаты анализа и работы по таскам хранятся в:

**`results/15-tasks-analysis/`**

Внутри — подпапки по ключу задачи (например `QAA-6369`), в них — отчёты, анализ, экспорты TestRail, скрипты и прочие артефакты по задаче.

Пример структуры:
```
results/15-tasks-analysis/
  QAA-6369/
    Survey_Spreadsheets_Test_Analysis.md
    Trajectory_Field_Clearing_Analysis.md
    testrail-section-6277-subtree-cases.json
    ...
```

## Маршрутизация

- если команда про **результаты анализа по таске** / **хранение артефактов работы по задаче** (Jira/QAA) → модуль `15-tasks-analysis`

## Быстрый старт (копипаст)

```
Открой: modules/15-tasks-analysis/README.md

Результаты работы по задаче QAA-6369 сохрани в results/15-tasks-analysis/QAA-6369/.
```
