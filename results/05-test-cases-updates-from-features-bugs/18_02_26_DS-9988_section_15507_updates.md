# План обновления тестов TestRail: DS-9988, секция 15507 (Table widget)

## 1. Контекст

**Jira:** [DS-9988](https://starsteer.atlassian.net/browse/DS-9988) — *Table. Trajectory. Display newly added trajectory points in the widget*

**Статус:** Closed (Fixed). Fix Version: M 2026.2.0

**Epic:** DS-8593 — Table. Trajectory

**TestRail:** проект 5, секция **15507** (Table widget).

---

## 2. Что изменилось по задаче

- **Автоотображение новых точек траектории** — новые точки траектории автоматически появляются в Table-виджете без перезагрузки страницы.
- **Умный автоскролл (followLastRow):**
  - Пользователь на последней строке → при поступлении новых данных таблица автоматически скроллится вниз.
  - Пользователь не на последней строке → положение скролла не меняется при поступлении новых данных.
- **Реализация:** кастомный автоскролл в TrajectoryTable заменён на встроенный в table-widget с параметром `followLastRow`.

**Связанные задачи:** DS-8596 (Table. Trajectory. Display data); DS-11669, DS-11734 — отдельные баги по отображению изменений точек с SS.

## 6. Новые тесты (рекомендации)

### 6.1 Trajectory: real-time обновление и умный автоскролл

- **Название:** Trajectory. Real-time data update with smart auto-scroll  
- **Refs:** DS-9988, DS-8593  
- **Предусловия:** Приватный дашборд, Table виджет, на виджет добавлен объект Trajectory с данными (достаточно строк для скролла). Возможность добавлять новые точки траектории в реальном времени (например, через StarSteer).  
- **Шаги и ожидания:**  
  1) Проскроллить таблицу вверх (последняя строка не видна). Добавить новую точку траектории. → Новая точка появляется без перезагрузки; позиция скролла не меняется.  
  2) Проскроллить вниз до последней строки. Добавить ещё одну точку. → Новая точка появляется, таблица автоматически скроллится вниз.  
  3) По желанию: перезагрузка страницы — все точки (в т.ч. добавленные в real-time) отображаются.

### 6.2 (Опционально) Trajectory в сервисной вкладке Spreadsheets

- **Название:** Trajectory. Real-time update in Spreadsheets tab  
- **Refs:** DS-9988  
- **Суть:** Проверка отображения новых точек траектории без перезагрузки во вкладке Spreadsheets (как в комментариях к задаче).
