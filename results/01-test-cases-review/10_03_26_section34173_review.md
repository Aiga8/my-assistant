## Секция 34173 — Window dropdown list

### 375660 — Displaying the add tab button.
- **Стиль (заголовок):** Точка в конце названия — в остальных кейсах секции тоже с точкой; можно унифицировать без точки при общем рефакторинге.
- Preconditions пустые; при желании добавить «Пользователь авторизован» для единообразия с 382331, 382336 и др.
- Логика кейса корректна.

### 377446 — Opening a dropdown by clicking the "+" button.
- В expected остались inline-стили (вставка из Confluence) — на исполнение не влияет.
- Замечаний по содержанию нет.

### 377447 — Display the "Public Window" tooltip when hovering over the icon.
- Шаг 1 дублирует ожидания 377446 (открытие меню, список, сортировка, иконка). Допустимо сократить до «Меню открыто» или оставить как есть для самодостаточности.
- Замечаний по смыслу нет.

### 377448 — Truncating long Window names with an ellipsis.
- Замечаний нет.

### 377449 — Displaying the full Window name in the tooltip when hovering over it.
- **Опечатка (expected шаг 2):** «полным именем » — лишний пробел в конце. Убрать.
- Шаг 1 совпадает с 377448 (ожидание трункации); можно не дублировать или сослаться на 377448.

### 377450 — The menu height adapts to the number of Windows.
- Замечаний нет.

### 377451 — Limiting the menu height when there are a large number of Windows.
- **Опечатка (preconds):** «много окон(больше» → «много окон (больше» (пробел перед скобкой).

### 377452 — Opening the Window in a new tab on click.
- Замечаний нет.

### 377453 — The menu closes automatically after selecting a Window.
- **Неполная формулировка (preconds):** «В меню есть доступные» — не указано что именно. Рекомендуется: «В меню есть доступные окна».

### 377454 — Displaying Windows that are already open.
- Замечаний нет.

### 377456 — The menu closes when you click outside the menu area.
- Замечаний нет.

### 377457 — Correct functioning of the open Window.
- Замечаний нет.

### 377458 — Switching between tabs closes the dropdown.
- Замечаний нет.

### 382331 — Sorting Windows alphabetically in dropdown menu
- Пересечение с 377446 (алфавитный порядок); здесь отдельно проверяется обновление списка после создания/переименования — сценарий корректен.
- Замечаний нет.

### 382336 — Menu positioning near screen edges
- В шаге 1 expected «Позиция установлена» звучит как констатация; при желании заменить на «Кнопка «+» у края экрана» для ясности. Некритично.
- Замечаний по смыслу нет.

### 382337 — Window icons consistency check
- Замечаний нет.

### 382338 — Menu behavior during Window operations
- **Формат (expected шаг 3):** Текст ошибки приведён на английском («Window has been removed», «Possible reasons...»). Если в продукте сообщение на EN — допустимо оставить; иначе привести к языку продукта. Двойное тире «— - Incorrect URL» → «— Incorrect URL» или маркированный список.
- Сценарий (удаление окна в другой вкладке, клик по удалённому, затем обновление меню) корректен.

### 382340 — Multiple clicks prevention on Window selection
- Замечаний нет.

### 382341 — Menu state persistence during tabset switching
- **Формулировка (шаг 2):** «Переключиться на пункт меню на другом табсете» — «пункт меню» легко понять как элемент выпадающего списка. Имеется в виду переключение на другой табсет (другую вкладку). Рекомендуется: «Переключиться на другой табсет (другую вкладку)» или «Кликнуть по области другого табсета».
- Остальная логика понятна.

## Рекомендации

### Правки в кейсах
- **377449:** убрать пробел в конце в «полным именем ».
- **377451:** в preconds добавить пробел: «окон (больше».
- **377453:** в preconds дописать: «В меню есть доступные **окна**».
- **382338:** в expected шага 3 поправить «— -» на «—» или список; при необходимости унифицировать язык сообщения об ошибке.
- **382341:** шаг 2 переформулировать: «Переключиться на другой табсет» вместо «на пункт меню на другом табсете».

### Консистентность
- Заголовки с точкой в конце (375660, 377446 и др.) — при желании убрать точки во всей секции.
- Preconditions: в части кейсов есть «Пользователь авторизован» / «Открыть Drillspot», в 375660 preconds пустые — можно выровнять.

### Дубликаты и пересечения
- Явных дубликатов нет. 377446 и 382331 оба проверяют алфавитную сортировку; 382331 добавляет проверку после создания/переименования — различие обосновано.
